File/Folder Browser for WP Backend

== Description ==

Browse files and Diectorys in WordPress backend

* Browse Files/Folder
* Download Files
* Edit Files with codepress
* ZIP Files and UNZIP Files
* Delete Files/Folder recursiv
* Copy Files/Folder recursiv
* Move Files/Folder
* Change premissions of Files/Folder
* Rename Files/Folder

Use it with most careful!!!

== Installation ==

1. Download FileBrowser Plugin.
1. Decompress and upload the contents of the archive into /wp-content/plugins/.
1. Activate the plugin through the 'Plugins' menu in WordPress
